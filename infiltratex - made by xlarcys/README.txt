Usage / Kullanım:

This script is designed for web vulnerability scanning and shell uploading.
Bu betik, web açıklarını taramak ve shell yüklemek için tasarlanmıştır.

Commands / Komutlar:
1. Site Açığı Tarama / Scan for Vulnerabilities
   - Enter the URL of the site you want to scan.
   - Taramak istediğiniz sitenin URL'sini girin.

2. Açığı Olan Sayfaya Shell Yükleme / Upload Shell to Vulnerable Page
   - Choose a shell type from the options provided.
   - Sunulan seçeneklerden bir shell türü seçin.
   - Enter the URL of the vulnerable page.
   - Açığı olan sayfanın URL'sini girin.

Logs / Kayıtlar:
- Results are saved in the 'logs' directory.
- Sonuçlar 'logs' dizinine kaydedilir.
- Shellerinizi 'shells' dizinine kayıtediniz.
- You have saved your shells in the 'shells' directory.

Make sure to run this script in an ethical manner and only on sites you have permission to test.
Bu beti etik bir şekilde çalıştırdığınızdan emin olun ve yalnızca test etme iznine sahip olduğunuz sitelerde kullanın.
